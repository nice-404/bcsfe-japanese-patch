# BCSFE 日本語ロケール導入ガイド

本リポジトリは、
[**bcsfe（Battle Cats Save File Editor）**](https://github.com/fieryhenry/BCSFE-Python) 用の日本語ロケールです。

---

## 導入手順

### 1. BCSFEを起動

---

### 2. 設定メニューを開く

以下の順番で選択してください。

```
7. Edit config
↓
3. Language
↓
5. Add Locale
```

---

### 3. ロケールURLを入力

以下のURLを入力して Enter を押します。

```
https://github.com/nice-404/bcsfe-japanese-patch
```

---

### 4. 確認

```
y
```

を選択します。

---

### 5. 再起動

BCSFEを再起動してください。

---

## 完了

再起動後、日本語化が適用されます。
